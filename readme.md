# 2020 Algorithm Study

## 1. 파일 올리는 방식
김건(이름)_8단계(단계명) 으로 폴더 생성 후 해당 폴더를 업로드 ex) 김건_8단계

## 2. 문제 푸는 방식
하루당 단계 하나씩 푸는 것으로 하되, 유동적으로

## 3. 알고리즘 관련 자료 있으면 올릴 것
깊이 우선 탐색 알고리즘 등과 같이 알고리즘 관련 좋은 자료 있으면 폴더 새로 생성 후 업로드
이는 폴더를 자기 이름으로 만든 후 학습 자료 게시
ex) 김건 폴더를 미리 생성 후, 관련 자료 있으면 해당 자료를 그 폴더에 넣는 것.
-> 폴더를 계속 만드는 것이 아니라 하나의 폴더만 사용할 것
